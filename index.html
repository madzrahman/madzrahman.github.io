Programming in Java 2021: 8-bit Sonic

For my Programming in Java module for Year 1, I created a project based on the classic game Sonic. I have created this with the use of City University's Physics Engine. This project utilizes the physics used in the engine with the use of physics-related methods such as setting velocity, changing gravity and also jumping. The project also has many implementations of sound clips, sophisticated character behaviour and Graphical User Interfaces (GUIs). The character's movement is controlled via keyboard.
